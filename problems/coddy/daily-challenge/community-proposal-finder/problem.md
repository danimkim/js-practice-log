Create a program that helps residents search through community proposals using autocomplete. Your program should read a list of proposal names and a search query, then display two things: first, all proposals containing the search term (in their original order), and second, only those matching proposals that start with a vowel (A, E, I, O, U), along with a count.

The vowel-filtered results should be sorted alphabetically. Separate the two sections with a blank line, and remember that vowel checking is case-insensitive.
